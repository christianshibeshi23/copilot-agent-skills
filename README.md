# Copilot Agent Skills

A collection of AI-powered skills for Microsoft Copilot Studio agents, with schema validation, templates, and AI-driven authoring support.

## Features

- Pre-built skill templates
- YAML-based agent definitions
- Schema validation
- AI-powered skill generation

## Usage

Import skills directly into your Copilot Studio environment or use the CLI tools for local development.
